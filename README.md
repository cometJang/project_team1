## 🔍 문제 정의

### 해결하고자 하는 문제
연애 예능의 흥행 요인은 주로 **정성적으로만 논의**되어 왔으며, 어떤 감정 흐름과 행동 선택이 시청자 반응을 증폭시키는지에 대한 **정량적 분석 사례가 부족**합니다.

### 문제의 중요성
- 감정 서사를 데이터로 구조화하여 콘텐츠 기획·편집·마케팅 전략에 활용 가능
- 시청자 과몰입 현상을 객관적 지표로 측정
- 차기 시즌 제작 전략 수립에 실증적 근거 제공

### 타겟 사용자
- 🎬 **OTT 콘텐츠 기획자 및 제작 PD** - 캐스팅, 편집 방향성 결정
- 📱 **유튜브·OTT 클립 마케팅 담당자** - 바이럴 클립 선정 및 배포 전략
- 📊 **미디어·콘텐츠 데이터 분석가** - 감정 서사 분석 방법론 연구

### 핵심 연구 질문

#### Q1. 성격 유형(MBTI)에 따라 이별 이후 감정 회복 패턴은 어떻게 다른가?
> MBTI 유형별 감정 타임라인 비교 분석

#### Q2. 감정 반전이 발생한 서사 구간에서 시청자 반응은 실제로 급증하는가?
> 감정 저점→반전 구간과 YouTube 조회수/댓글 상관관계 분석

#### Q3. 관계 선택을 번복하는 출연자는 더 부정적인 여론에 노출되는가?
> 선택 번복 횟수와 부정 댓글 비율 상관관계 분석

## 📊 데이터 소스

### 데이터 출처

#### 1. 방송 기반 데이터
- **출처**: 나무위키, 방송 요약 자료, 자막 기반 수작업 태깅
- **내용**: 
  - 출연자 프로필(MBTI, 나이, 성별)
  - 회차별 주요 사건
  - 감정 변화 로그
  - 문자 발송 기록

#### 2. 시청자 반응 데이터
- **출처**: [YouTube Data API v3](https://developers.google.com/youtube/v3)
- **내용**: 
  - 환승연애2 관련 클립 영상 메타데이터
  - 조회수, 좋아요 수, 댓글 수
  - 댓글 텍스트 데이터

### 수집 방식
- **스냅샷 방식(1회 수집)**
- 종영 콘텐츠 특성을 고려하여 수집 기준일의 데이터로 고정
- API 할당량 범위 내에서 효율적 수집 전략 수립

### 데이터 볼륨

| 데이터 유형 | 예상 볼륨 | 테이블 |
|------------|-----------|--------|
| 출연자 메타데이터 | 약 10명 | `character_profile` |
| 회차·사건·감정 로그 | 약 150~200 rows | `episode_event_log`, `emotional_timeline` |
| 문자 발송 로그 | 약 120~180 rows | `message_log` |
| 유튜브 영상 | 약 20~50개 | `youtube_video_metrics` |
| 유튜브 댓글 | 약 2,000~3,000 rows | 샘플링 또는 상위 N개 |

### 데이터 형식
- **YouTube Data API v3**: JSON
- **방송 기반 데이터**: CSV
- **데이터베이스**: Supabase (PostgreSQL)
